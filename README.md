# Pandoc-GUI


pandoc 3.8.1 下载链接：[https://github.com/jgm/pandoc/releases/tag/3.8.1]

本项目安装的版本是 pandoc-3.8.1-windows-x86_64.zip

本项目使用的核心技术：GUI框架PyQt5、核心引擎Pandoc、部署工具Pyinstaller、Python 3.12.10（64位）

目标实现功能：
1. 上传文件，将文件转换为其他格式。
2. 上传文件后，可选择上传排版模板docx文件，将原文件内容转换格式的同时，根据模板文件进行排版的修改。

工作流程是：

用户点击"排版配置"按钮或菜单项，打开排版配置对话框
用户在对话框中配置各种排版参数（字体、段落、页面等）
用户可以点击"导出模板"按钮，将当前配置导出为Word模板文件
用户可以使用"选择模板文件"功能选择这个导出的模板文件
用户使用pandoc转换功能，将文档转换为带有自定义排版的格式


基础文本与段落样式：段落、正文、正文字符（包含中文、英文、数字的字体）
标题与层级样式：标题1、标题2、标题3、标题4、标题5、标题6、标题7、标题8、标题9
列表样式：有序列表、无序列表
引用与交互元素样式：超链接、脚注、脚注引用文本
非文本与布局样式块：块文本、水平线、表标题、表格、图片说明
